# Commands are case sensitive and are in lower case.

## Login to Azure
    az login

## commands to list resources
    az storage account list

    az group list

    az vm list

## Create a resource group if you don't have one already. [Reference](https://docs.microsoft.com/en-us/cli/azure/group?view=azure-cli-latest)
    az group create --name cligroup --location EastUs

# Create a Vm [Rerefence](https://docs.microsoft.com/en-us/cli/azure/vm?view=azure-cli-latest)

    az vm create --resource-group cligroup --name aznewvm --image win2016datacenter --admin-username  bingo
    
# Show the details for a managed Kubernetes cluster.. [Reference](https://docs.microsoft.com/en-us/cli/azure/aks?view=azure-cli-latest#az_aks_show)
    az aks show
